Game Overview

Welcome to our game! This is a unique 3D experience where players control a rolling ball through a procedurally generated world. The game world is created from seeds, which are generated based on color sequences.

The game features:

  • Dynamic world generation with color-based seeds
  • Interactive chunk system that changes the game environment
  • AI-controlled balls that chase the player
  • Multiplayer functionality using Bluetooth for local connections
  • Multiple levels to explore, each with increasing difficulty

Game Features

  • Customizable world generation
  • Strategic gameplay with AI opponents
  • Color-based interactions to influence the environment
  • Fast-paced multiplayer with local connections
  • Progressive level difficulty with unique challenges

Game Development

This game is developed by Bjarnik Interactive, a small team focused on delivering unique and innovative gaming experiences. The game is built using Unity and incorporates cutting-edge features like AI-driven gameplay, multiplayer, and procedural generation.
